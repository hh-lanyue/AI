1 修复 pip 命令
python -m ensurepip
python -m pip install --upgrade pip
pip install --upgrade pip



2 批量卸载包
2.1 创建文件名 info.txt
2.2 将要卸载的包按每个一行写入到 info.txt 中
2.3 pip uninstall -r info.txt



3 批量安装包
3.1 创建文件名 info.txt
3.2 将要卸载的包及版本号按每个一行写入到 info.txt 中
3.3 pip install -r info.txt

eg:
dataclasses==0.8
numpy==1.19.5
pip==21.3.1
torch==1.10.2
torchvision==0.11.3




4 更新 pip
pip install --upgrade pip
pip install -U pip



5 卸载单个程序包
pip uninstall [包名]



6 更新单个程序包
pip install --upgrade [包名]
pip install -U [包名]



7 指定安装源安装【一次性】
pip install [包名] -i [安装源地址]



8 指定安装源安装【永久性】
pip config set global.index-url [安装源地址]

eg:
清华源
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple/



9 下载包但是不安装
pip download [包名] -d [路径]



10 安装指定版本的包
pip install [包名]==[版本号]
